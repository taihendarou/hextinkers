É isso aí! Uma grande atualização para o TileBravo.

Para quem não conhece, o TileBravo é um editor de tiles que desenvolvo, com o objetivo de cobrir os principais recursos necessários para editar gráficos em projetos de ROM hacking.

E agora o **TileBravo** acaba de receber a atualização **1.1.0**.

### Principal novo recurso: Editor hexadecimal

Além de atualizações de desempenho, o principal novo recurso é um editor hexadecimal embutido. Isso facilita dump de vram, edições rápidas, depuração etc.

- Abra o editor hexadecimal pressionando o atalho "H"
- Você visualizará os bytes correspondentes aos tiles do gráfico carregado no editor
- Você pode colar bytes copiados de outro editor ou do emulador
- Ao clicar em qualquer tile, os bytes equivalentes recebem um highlight
- Ao editar valores de bytes, os tiles atualizam em tempo real
